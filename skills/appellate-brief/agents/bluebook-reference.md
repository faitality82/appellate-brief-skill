# Bluebook Citation Reference — Michigan Practice

## Cases (Bluebook Rule 10)

### Full Citation (first use in a section)
```
In re Sours, 459 Mich 624, 630; 593 NW2d 520 (1999).
People v Smith, 312 Mich App 201, 210; 877 NW2d 484 (2015).
```

**Components:** `[Party v Party], [vol] [reporter] [first page], [pinpoint]; [vol] [regional reporter] [first page] ([year]).`

### Short Citation (subsequent uses)
```
Sours, 459 Mich at 630.
Smith, 312 Mich App at 210.
Id. at 631.    ← only when immediately following same source
```

### Michigan Reporters
| Reporter | Court | Example |
|---|---|---|
| Mich | Michigan Supreme Court | 459 Mich 624 |
| Mich App | Michigan Court of Appeals | 312 Mich App 201 |
| NW2d | Regional (parallel cite required) | 593 NW2d 520 |

### Parenthetical Explanations
```
In re Trejo, 462 Mich 341, 350; 612 NW2d 407 (2000) (holding that statutory
grounds must be proven by clear and convincing evidence).
```

### Unpublished Opinions
```
People v Jones, unpublished per curiam opinion of the Court of Appeals,
issued May 15, 2023 (Docket No. 361234).
```
Not binding but citable as persuasive authority per MCR 7.215(C)(1).

## Statutes (Bluebook Rule 12)

### Format
```
MCL 712A.19b(3)(b)(i).
MCL 722.623(1)(a).
```

**Components:** `MCL [title].[section]([subsection])([sub-subsection]).`

### Quoting Statutory Language
```
MCL 712A.19b(3)(g) provides that termination is warranted when "the parent,
without regard to intent, fails to provide proper care..."
```

### Amended Statutes
```
MCL 712A.19b (as amended by 2018 PA 58).
```

## Signals (Bluebook Rule 1.2)

| Signal | Meaning |
|---|---|
| [no signal] | Direct support |
| See | Implicit support — clearly supports but not directly |
| See also | Additional support |
| Cf. | Analogous support |
| But see | Authority supports contrary position |
| See generally | Background / general reference |

Introductory signals are italicized in the brief.

---

## Citation Placement

| Type | Format |
|---|---|
| Citation sentence | Period after. `"The court held X. Smith, 312 Mich App at 210."` |
| Citation clause | Comma separation. `"As held in Smith, 312 Mich App at 210, the standard..."` |
| String citations | Semicolons between. `"Smith, 312 Mich App at 210; Jones, 459 Mich at 630."` |

---

## Michigan-Specific Rules

1. **Parallel citations required** — always include both Mich/Mich App AND NW2d
2. **Pinpoint cites required** for all quotations and specific propositions
3. **"In re" cases** — italicize "In re" in the brief
4. **Unpublished opinions** — use full "unpublished per curiam opinion" designation
5. **MCR 7.215(C)(1)** — unpublished opinions not binding, may be persuasive only
